Editor do post

O arquivo .md será nomeado a partir do campo title. A data pode ser digitada em formato brasileiro, como 15/04/2025.

O cabeçalho será gerado como yyyy-mm-dd.
arquivo: post.md
download: navegador decide a pasta padrão
No Windows, o arquivo normalmente vai para a pasta Downloads. Em alguns navegadores modernos, pode aparecer uma janela para escolher onde salvar.

Preview

Aqui você confere o conteúdo antes de baixar. A aba de visualização mostra uma leitura do post e a aba de Markdown bruto mostra o texto exato do arquivo.
